How It Works

Three steps. Zero config.

From install to wiki in under 2 minutes.

1

Install the skill

npx skills add susumutomita/codewiki
One-time setup. Adds the /codewiki command to your Claude Code.

2

Run /codewiki

In any project, open Claude Code and type /codewiki. Claude scans your entire codebase, reads every key file, and writes a deep analysis of each component.

3

Explore & Ask

A local web server opens in your browser. Browse the generated wiki, read the analysis, and ask follow-up questions in real-time via the chat widget or right-click menu.
